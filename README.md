# Ansh Battery Care Website

A professional React-based website developed for a battery and inverter retail business located in Sangaria, Hanumangarh, Rajasthan.

## 🚀 Features
- Clean and modern UI
- Product showcase (Car, Bike, Inverter batteries)
- Battery Finder tool for basic recommendations
- WhatsApp enquiry integration
- Fully responsive design

## 🛠 Tech Stack
- React (Vite)
- JavaScript
- HTML & CSS

## 📍 Business Details
**Ansh Battery Care**  
Sangaria, Hanumangarh, Rajasthan

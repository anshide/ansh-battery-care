# Battery & Inverter Retail Website

A professional React-based web application developed for a battery and inverter retail business.

## 🚀 Features
- **Clean & Modern UI**: A visually appealing interface with a dark blue and emerald green color palette.
- **Product Showcase**: Categorized catalog for Car, Bike, and Inverter batteries.
- **Dynamic Battery Finder**: A smart tool to filter batteries by vehicle type, brand, model, and fuel type.
- **Digital Invoice Request**: A dedicated section to seamlessly request bills/invoices.
- **WhatsApp Integration**: Instant redirection for product enquiries and invoice requests directly via WhatsApp.
- **Fully Responsive**: Optimized for both desktop and mobile viewing.

## 🛠 Tech Stack
- React (Vite)
- JavaScript
- HTML5 & CSS3

## ▶️ Run Locally

1. Install the project dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```
